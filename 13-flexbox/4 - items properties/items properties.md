# Propriedades para os itens

- flex-basis
- flex-grow
- flex-shrink
- flex
- order

## flex-grow

- O crescimento do item dentro do caontainer em relação aos espaços vazios.

## flex-shrink

- O encolher do item dentro do container.

## flex 

- shorthand
- flex-grow flex-shrink flex-basis
- podem ter 1, 2 ou 3 valores
